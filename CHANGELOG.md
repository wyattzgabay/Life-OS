## [0.1.0-alpha] - 2024-12-07

### Added
- Initial alpha release
- Workout programming with weekly schedule
- Volume tracking (MEV/MAV/MRV per muscle group)
- Double progression system for lifting
- Running programs (5K, 10K, Half Marathon, Marathon)
- VDOT-based pace calculator
- AI food logging via Groq API
- XP and leveling system (10 levels)
- Streak tracking with multipliers
- Firebase authentication (email/password)
- Cloud sync with Firestore
- Demo mode for showcasing
- Admin panel for feedback review
- Exercise swap system
- Running injury protocols (6 injuries)
- Daily scoring system
- Weekly stats view

### Known Issues
- 58 console.log statements need removal
- State.js is too large (1900+ lines)
- No offline mode
- No data export
- Limited error handling
